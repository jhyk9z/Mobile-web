# 정보 기말 JB

남색 배경의 단일 로그인 화면입니다.

## 포함 기능

- 로그인 / 회원가입 전환
- 회원가입 계정 저장
- 로그인 세션 데모 저장
- 로그인 후 단원 목록 화면 전환
- 로그아웃

## 데이터 저장 방식

현재 버전은 GitHub Pages 같은 정적 호스팅에서도 동작하도록 브라우저의 `localStorage`에 계정과 로그인 세션을 저장합니다.

이 방식은 데모용입니다. 실제 서비스에서는 비밀번호를 브라우저에 저장하면 안 되며, 서버 인증으로 교체해야 합니다.

## 실행

브라우저에서 `index.html`을 열면 바로 확인할 수 있습니다.
