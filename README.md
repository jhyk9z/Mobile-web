# Mobile Web

모바일 우선 회원 웹사이트 시작 템플릿입니다.

## 포함 기능

- 회원가입 화면
- 로그인 화면
- 로그인한 사용자별 프로필 상태 저장
- 로그인한 사용자별 개인 메모 저장
- 모바일 우선 반응형 디자인

## 데이터 저장 방식

현재 버전은 GitHub Pages 같은 정적 호스팅에서도 동작하도록 브라우저의 `localStorage`에 데이터를 저장합니다.

이 방식은 데모와 디자인 검증용입니다. 실제 서비스에서는 비밀번호를 브라우저에 저장하면 안 되며, Supabase, Firebase, 자체 API 서버 같은 백엔드 인증과 데이터베이스로 교체해야 합니다.

## 실행

브라우저에서 `index.html`을 열면 바로 확인할 수 있습니다.
