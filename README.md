# 정보 기말 JB

남색 배경의 단일 로그인 화면입니다.

## 포함 기능

- 로그인 / 회원가입 전환
- Supabase Auth 회원가입
- Supabase Auth 로그인 세션 유지
- 로그인 후 학습 앱 화면 전환
- 단원 선택
- 개념 카드 / 용어 카드 / 퀴즈 탭
- 단원별 진행률 저장
- 로그아웃

## 데이터 저장 방식

현재 버전은 Supabase Auth로 회원가입, 로그인, 로그아웃을 처리합니다. 브라우저에는 Supabase 세션만 저장되고 비밀번호는 저장하지 않습니다.

## 실행

브라우저에서 `index.html`을 열면 바로 확인할 수 있습니다.
